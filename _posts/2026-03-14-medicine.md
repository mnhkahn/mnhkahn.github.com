### 说明

本应用用于帮助用户快速掌握药品管理、每日提醒、历史记录、数据备份等全部功能。
所有数据均保存在手机本地，不联网、不上传。

| 功能模块     | 说明                                    |
| ------------ | --------------------------------------- |
| 药品管理     | 添加 / 删除药品名称、剂量、每日提醒时间 |
| 每日定时提醒 | 按设定时间准时提醒，每天自动重复        |
| 服药历史记录 | 查看所有已提醒、已确认服药的记录        |

### 使用指南

#### 添加药品

1. 点击主界面右下角 + 按钮
2. 填写：
   - 药品名称（必填）
   - 服药剂量（可选）
3. 点击 选择时间，设置每日提醒时刻
4. 点击 保存
5. 系统自动设置每日重复闹钟

#### 删除药品

1. 在药品列表找到对应药品
2. 点击删除按钮
3. 系统自动取消该药品的所有提醒
#### 查看服药历史

1. 点击主界面 右上角菜单（三个点）
2. 选择 服药历史
3. 可查看：
   - 药品名称
   - 提醒时间
   - 服药状态（已服药）

#### 常见问题 FAQ

1. Q1：提醒只响一次，之后不提醒？
A：通常是被系统杀后台。
请开启：自启动 + 后台运行 + 关闭电池优化。
2. Q2：收不到通知？
A：检查：
APP 通知权限是否开启
手机是否开启静音 / 勿扰模式
3. Q3：APP 升级后数据不见了？
A：正常覆盖升级不会丢数据。
如丢失，点击 恢复数据 即可从备份找回。
4. Q4：手机重启后提醒失效？
A：不会。
APP 已支持开机自动恢复所有闹钟。

#### 注意事项

1. 卸载 APP 会清空本地数据，卸载前请务必备份。
2. 提醒依赖系统闹钟，请勿使用 “清理内存” 类工具强制关闭 APP。
3. 所有数据仅存在本地，隐私安全。
4. 最低支持 Android 8.0（API 26）。
